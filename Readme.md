### Development

1. install the dependencies with
   `pip3 install Pillow pyautogui pytesseract matplotlib opencv-python`
2. virtual env set
   `soure venv/binactivate`
3. run project
   `./dist/main/`

### Scrape database

1. install the dependencies with
   `pip3 install beautifulsoup4 requests`
2. virtual env set
   `soure venv/binactivate`
3. run project
   `python3 scrape.py`
